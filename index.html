ℹ️ How it works

The official GPX track is loaded from vienna-marathon.com. Wind data is fetched from Open-Meteo (free, no API key). The path integral computes the dot product of the wind vector with the running direction at each track segment, then averages over the total distance. Positive = net tailwind, Negative = net headwind. The heatmap on the map shows green for tailwind and red for headwind segments.
