ℹ️ How it works

The GPX track is loaded from a local file included in this repository. Wind data is fetched from Open-Meteo (free, no API key). Specify your expected runtime so that wind changes during the race are accounted for – the tool interpolates 15-minute forecasts based on your pace. The path integral computes the dot product of the wind vector with the running direction at each track segment, then averages over the total distance. Positive = net tailwind, Negative = net headwind. The heatmap on the map shows green for tailwind and red for headwind segments.
